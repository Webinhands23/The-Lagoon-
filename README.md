# The Lagoon 🌊

**The Lagoon Digital** - A modern restaurant & lounge website built with Lovable.

## About
The Lagoon is a premium waterfront dining and lifestyle brand. This site showcases our menu, events, reservations, and brand story.

## Features
- **Responsive Design**: Works on phone, tablet, and desktop
- **Menu Showcase**: Food, drinks, and specials section  
- **Event Booking**: Reserve tables and private events
- **Brand Story**: About The Lagoon experience
- **Contact Info**: Location, hours, and social links
